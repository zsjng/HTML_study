HTML CSS

CSS is used to format the layout of a page.

With CSS, you can control the color, font, the size of text, the spacing between elements,
how elements are positioned and laid out, what background images or background colors are to be used,
different displays for different devices and screen sizes, and much more!

CSS can be added to HTML in 3 ways:

Inline CSS: use style attribute inside HTML elements.

Internal CSS: use <style> element in the <head> section.

External CSS: use a <link> element to link a CSS file.

Inline CSS

Inline CSS can just change the single HTML element. 

    <p style="color: red">
        This is the only one HTML element with color red.
    </p>

This is the only one HTML element with color red.

Internal CSS

    <!DOCSTYLE html>
    <html>
    <head>
        <style>
            body {background-color: powderblue;}
            h1   {color: blue;}
            p    {color: red;}
        </style>
    </head>
    </html>

External CSS

    <!DOCTYPE html>
    <html>
    <head>
      <link rel="CSS_file" href="CSS_file.css">
    </head>
    <body>
        <h1>This is a heading</h1>
        <p>This is a paragraph.</p>
    </body>
    </html>

Here is what external CSS file look like: 

    body {
      background-color: green;
    }
    h1 {
      color: yellow;
    }
    p {
      color: blue;
    }
