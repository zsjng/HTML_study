HTML styles

The HTML style attributes is used to add styles to an element.
such as color, font, size, and more.

Color and Font-size

I am red color 

    <p style="color:red;font-size: 60px;">
        I am red color
    </p>

The HTML style attribute has the following syntax:

<tagname style="property:value;">

Background Color

in this HTML, I use follow code to change the page background color: 

    <h3 style="background-color: darkgreen;"> This is title with background color:darkgreen.</h3>

This is title with background color:darkgreen.

    <p style="font-family: 'Bernard MT Condensed',ui-monospace;font-size: x-large;background-color: tomato;">
        This is paragraph with fonts:'Bernard MT Condensed', size: x-large, background color:tomato.
    </p>

This is paragraph with fonts:'Bernard MT Condensed', size: x-large, background color:tomato, color : darkgreen.

Text fonts, size, color can use together.

Text Alignment

The CSS text-align property defines the horizontal text alignment for an HTML element: 

    <p style="text-align: center">
        This is center
    </p>

This is center 

    <p style="text-align: right">
        This is right
    </p>

This is right
