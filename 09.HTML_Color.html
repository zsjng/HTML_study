HTML Color

HTML color can be specified by using color name, or RGB, HEX, RGBA,etc.

Color name

    <h2 style="background-color: red">red</h2>
    <h2 style="background-color: orange">orange</h2>
    <h2 style="background-color: yellow">yellow</h2>
    <h2 style="background-color: rgba(255,70,50,0.9)">rgba(255,70,50,0.9)</h2>
    <h2 style="background-color: hsl(5,20%,70%)">hsl(5,20%,70%)</h2>
    <h2 style="background-color: #005544;color: aliceblue">#005544</h2>
    <h2 style="background-color: rgb(255,60,84)">rgb(255,60,84)</h2>
