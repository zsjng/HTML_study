HTML Attributes

The href Attribute

    <a href="https://github.com/zsjng">This is my GitHub </a>

 This is my GitHub

 <a> tells browser this is a hyperlink.

 href specifies the URL of the page the link goes to.

The src Attribute

    <img src="my_logo.jpg" alt="this is my local image" width="125" height="125">

 

 img tell browser this is a image.

 src tell browser where the image is.

 alt will replace the image when the image can not show.

 width and height are very easy to understand.

The lang Attribute

    <html lange='xx-YY'>.

 xx mean what language you use. like en or zh.

 -YY is optional, just add your country code like US or CN.

The title Attribute

    <p title="when you move your mouse to here, you can see it">

 move your mouse to here.

 title when you move your mouse to element, you can see it.
